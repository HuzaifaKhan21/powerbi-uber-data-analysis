# Uber Data Analysis Dashboard (Power BI)

## Tools Used
- Power BI
- Power Query
- DAX

## Project Overview
This project analyzes Uber ride data to understand ride completion, cancellations, revenue trends, distance traveled, and vehicle performance.

## Key Insights
- Completed rides significantly outperform cancelled rides
- Revenue varies by vehicle type and location
- Monthly trends highlight peak demand periods
- Customer ratings help identify service quality patterns

## Dashboard
The Power BI dashboard includes KPI cards, trend analysis, vehicle-wise performance, and interactive filters.

## Status
Project completed and ready for business use.
